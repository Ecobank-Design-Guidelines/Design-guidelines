Principles

Design



Principles enable designers and developers to speak the same language during idea generation and quality checking of designs.

Overview

  • 0.1 Provide feedback in context
  • 0.2 Display secondary content independently
  • 0.3 Show users only what’s necessary
  • 0.4 Prevention is better than cure
  • 0.5 Help users recover from errors
  • 0.6 Edit in context
  • 0.7 Provide a sense of control

Provide feedback in context

Give the user feedback that’s relevant to the specific content area on the page.

  • • Alert line
  • • Alert Block
  • • Note

Display secondary content independently

Display secondary content, which doesn’t directly relate to the primary content, independently. E.g. warning messages.

  • Modal

Show users only what’s necessary

Only reveal secondary, related features or content if users ask for them. This is called progressive disclosure.

  • • Buttons
  • • Checkbox
  • • Hyperlink
  • • Radio buttons
  • • Search select
  • • Select
  • • Toggle
  • • Tooltip

Prevention is better than cure

Rather than relying on error messages, prevent errors in the first place.

  • • Alert inline
  • • Alert block
  • • File upload /li>
  • • Currency input
  • • Date input
  • • Input
  • • Number scrubber
  • • Password input
  • • Phone number input
  • • Note
  • • Search select
  • • Select
  • • Tooltip

Be specific to the user

When writing error messages, ensure that the copy is customized to the actual error that occurred and how to revive from it.

  • • Alert inline
  • • Alert block
  • • Content states
  • • Currency input
  • • Input
  • • Number scrubber
  • • Password input
  • • Phone number input
  • • Search select
  • • Select

Edit in context

Allow users to edit content in-page and receive immediate feedback. Don’t take them to a separate page.

  • • Buttons
  • • Hyperlinks

Provide a sense of control

Give the user a view of what to expect and options to leave.

  • • Pagination
  • • Progress bar
  • • Stepper

W3C World Wide Web Consortium is an international organization committed to improving the web.
