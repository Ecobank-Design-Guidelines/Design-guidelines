Link to Sketch libraries

We have our own design library that contains almost everything you need to build out your products. Follow these steps to get the library linked to your Sketch.


  • 0.1 Open Sketch,
  • 0.2 Go to Sketch/Preferences/Libraries or press ⌘+,
  • 0.3 Click on “Add Library”,
  • 0.4 Navigate to your Dropbox folder and select (Nedbank Sketch Library)

Using the Library

We have our own design library that contains almost everything you need to build out your products. Follow these steps to get the library linked to your Sketch.


  • 0.1 Download the Base template.
  • 0.2 You’ll see 3 artboards (Mobile, Tablet and Desktop).
  • 0.3 Each artboard has a reference of how the grid is constructed you can hide this layer, and hit Ctrl + L to show the layout guides.
  • 0.4 To demonstrate how to use the components with the layout grid, lets use the Desktop artboard. To focus in on just the Desktop artboard click on it and hit cmd ⌘ +2.
  • 0.5 Click on insert/Nedbank - Web Components/Button/Primary/Large and place the component on your artboard and align the Button to the left side of the very first column.
  • 0.6 Most components are responsive on the x-axis so you can resize them while keeping their intended use. Lets adjust the Button to the right side inside the fourth column.
  • 0.7 Next, if you look at the right panel, you’ll see a section with overrides, you’ll see an input box with the label Primary, Type is “New Button” and press enter.
  • 0.8 Most components behave in exactly the same way, add a few more and play with the overrides to familiarize yourself a bit more.
